# Birdnest

## App description

Birdnest implementation for the Reaktor assignment. Brief available at <https://assignments.reaktor.com/birdnest/>.

## Author

Niko Suoniemi <niko@tamperelainen.org>

## App URL

<https://pfnisu-birdnest.onrender.com/>

## Technology

* Backend
    * Node.js 18
    * Express 4
    * PostgreSQL
* Frontend
    * Vanilla Web Standards
